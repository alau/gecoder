    expect_constant_set(constant_set) click to toggle source

    Returns the arguments that should be used in a partial mock to expect the specified constant set (possibly an array of arguments).

    # File specs/spec_helper.rb, line 86
def expect_constant_set(constant_set)
  if constant_set.kind_of? Range
    return constant_set.first, constant_set.last
  elsif constant_set.kind_of? Fixnum
    constant_set
  else
    an_instance_of(Gecode::Raw::IntSet)
  end
end
