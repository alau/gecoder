Communication

The following are archives of the mailing lists that used to be hosted on RubyForge (which has since shut down):

gecoder-users
Usage questions and general Gecode/R discussion belongs here. Basically anything about Gecode/R that doesn’t belong in gecoder-devel goes here.
gecoder-devel
Discussion about the development of Gecode/R goes here.

Git
