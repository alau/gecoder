Details

The project is hosted on RubyForge, which also hosts the project’s mailing lists, version control and issue tracker.

Communication

The following mailing lists are used for communication.

gecoder-users
Usage questions and general Gecode/R discussion belongs here. Basically anything about Gecode/R that doesn’t belong in gecoder-devel goes here.
gecoder-devel
Discussion about the development of Gecode/R goes here.
gecoder-commits
Diffs of all commits to the subversion repository are sent here. Reviewing the code changes and sending feedback to gecoder-devel is encouraged.

License

Gecode/R is licensed under the GNU LGPL. Gecode is licensed under the MIT license.

Subversion

Gecode/R uses subversion for revision control. The repository can be read anonymously via http and checked out using

svn checkout svn://rubyforge.org/var/svn/gecoder/trunk gecoder

Issue Tracker

The issue tracker is available from the RubyForge project page. It handles bugs, feature requests and patches.
